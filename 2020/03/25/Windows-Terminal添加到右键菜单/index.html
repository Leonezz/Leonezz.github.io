Windows Terminal添加到右键菜单

发表于 2020-03-25 更新于 2020-03-28 阅读次数： 阅读次数：

写在前面

经常使用命令行进行一些操作，碍于Windows cmd丑陋的黑框再加上有高可定制化高颜值的Window Terminal的存在，导致我一度手动点开然后cd xxx/xxx到想打开的路径，而不是在Windows的资源管理器导航栏里输入cmd了事。可能这就是倔强吧😁
假如可以在文件夹里面快捷打开Terminal并进入路径的方法就好了。

把Windows Terminal快捷方式添加到右键菜单

可以通过添加注册表的方式实现。

  1. 首先，检查常量%USERPROFILE%和%LOCALAPPDATA%的值，可以通过在命令行输入echo %USERPROFILE%和echo %LOCALAPPDATA%来检查。如果没有报错的话说明正常。(正常不会有问题)

  2. 在命令行输入命令 mkdir "%USERPROFILE%\AppData\Local\terminal"创建文件夹，然后找个好看的图标(ico)文件放进去，它将出现在右键菜单的头部作为图标。如果找不到好康的，也可以到这里下载Windows Terminal默认的图标。

  3. 新建文件wt.reg，把下面的代码写入保存：

    Windows Registry Editor Version 5.00

[HKEY_CLASSES_ROOT\Directory\Background\shell\wt]
@="Open terminal here"
"Icon"="%USERPROFILE%\\AppData\\Local\\terminal\\terminal.ico"
[HKEY_CLASSES_ROOT\Directory\Background\shell\wt\command]
@="C:\\Users\\{your user name}\\AppData\\Local\\Microsoft\\WindowsApps\\wt.exe"

    其中第一个@处是将要出现在右键菜单的提示语
    它的下一句是图标的路径
    最后一句是Windows Terminal的程序路径，他需要是绝对路径，因此请把{your user name}换成你的Windows用户名。

    双击运行，添加注册表。

  4. 在任意文件夹点击右键，可以发现Open Terminal here已经出现在右键菜单。

  5. 点击该右键菜单，如果cmd当前目录仍然是C:/Users/{your user name}，请打开Windows Terminal的配置文件profile.json，把第一个CMD的startingDirectory字段删除。
